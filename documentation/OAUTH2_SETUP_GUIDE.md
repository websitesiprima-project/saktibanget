# Setup Google Drive dengan OAuth2 (untuk Personal Account)

## Masalah yang Ditemukan
Service Account **TIDAK BISA** digunakan untuk upload ke personal Google Drive (My Drive).
Error: "Service Accounts do not have storage quota"

## Solusi: Gunakan OAuth2

### LANGKAH 1: Buat OAuth2 Credentials di Google Cloud Console

1. **Buka Google Cloud Console**
   - https://console.cloud.google.com/

2. **Pilih Project** (yang sudah ada atau buat baru)

3. **Enable Google Drive API** (jika belum)
   - Menu: APIs & Services > Library
   - Cari "Google Drive API"
   - Klik ENABLE

4. **Configure OAuth Consent Screen**
   - Menu: APIs & Services > OAuth consent screen
   - User Type: **External** (untuk personal account)
   - Klik CREATE
   
   **App Information:**
   - App name: `PLN SAKTI Upload System`
   - User support email: (email Anda)
   - Developer contact: (email Anda)
   - Klik SAVE AND CONTINUE
   
   **Scopes:**
   - Klik ADD OR REMOVE SCOPES
   - Cari dan pilih: `../auth/drive.file`
   - Klik UPDATE > SAVE AND CONTINUE
   
   **Test users:**
   - ADD USERS
   - Masukkan email Google Anda
   - Klik SAVE AND CONTINUE

5. **Buat OAuth2 Client ID**
   - Menu: APIs & Services > Credentials
   - CREATE CREDENTIALS > OAuth client ID
   - Application type: **Desktop app**
   - Name: `PLN SAKTI Desktop Client`
   - Klik CREATE
   
   **Download JSON:**
   - Klik tombol download (icon ⬇)
   - Save as: `credentials.json`
   - Copy file ini ke: `d:\MAGANG\MAGANG-PLN\vlaas-pln\src\services\server\`

### LANGKAH 2: Generate Token

Jalankan script berikut untuk generate token:

```bash
node scripts/generate-drive-token.js
```

Script ini akan:
1. Buka browser
2. Minta Anda login dengan Google account
3. Minta izin akses Google Drive
4. Generate file `token.json`

### LANGKAH 3: Test Connection

```bash
node test-gdrive-connection-oauth.js
```

### LANGKAH 4: Restart Development Server

```bash
npm run dev
```

## File yang Dibutuhkan

1. `credentials.json` - OAuth2 client credentials (dari Google Cloud Console)
2. `token.json` - Access token (generated by script)

## Struktur Folder

```
src/services/server/
├── credentials.json     ← OAuth2 credentials (download dari GCP)
├── token.json          ← Generated after authorization
└── server.cjs
```

## Keamanan

- JANGAN commit `credentials.json` dan `token.json` ke Git
- Tambahkan ke `.gitignore`:
  ```
  src/services/server/credentials.json
  src/services/server/token.json
  ```
